About the Making of Copy Pasta

Why not React or any other framework, you ask?

Well, for a two page web application it didn't warrant a large library. I thought about it to show that I could set it up, but it would have been overkill.

Why assets folder?

It is easier to find the things that you want this way.

Why a separate JavaScript file instead of a script in the HTML page?

Because...

  1. For testing purposes (which I have none here)
  2. Ability to minify the file
  3. Reduces the size of the HTML

Why all of these very specific tags?

It is:

  1. Great for code readability
  2. Easier to read for the user
  3. Great for SEO if you want people to find your website
